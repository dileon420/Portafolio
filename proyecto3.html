E-commerce para Tienda de Ropa

Este proyecto es una plataforma de comercio electrónico desarrollada con Java Spring Boot y Thymeleaf, diseñada para una tienda de ropa en línea. Utiliza una base de datos MySQL para gestionar productos, usuarios y pedidos. Además, implementa roles de usuario y administrador para controlar el acceso y las funcionalidades disponibles.

Características Principales:

  • Catálogo de Productos: Visualización de productos con detalles, imágenes y categorías.
  • Carrito de Compras: Los usuarios pueden agregar productos al carrito y gestionar las cantidades.
  • Gestión de Usuarios: Registro y autenticación de usuarios con roles de administrador y cliente.
  • Panel de Administrador: El administrador puede agregar, editar y eliminar productos, así como gestionar pedidos y usuarios.
  • Seguridad: Implementación de Spring Security para la gestión de roles y protección de rutas.
