Tienda en Línea Interactiva

Este proyecto es una tienda en línea interactiva que permite a los usuarios navegar por productos, agregar artículos a un carrito de compras y proceder a la compra. El sitio web utiliza HTML, CSS y JavaScript para crear una experiencia visualmente atractiva y funcional.

Características Principales:

  • Carrito de Compras: Los usuarios pueden agregar productos a un carrito dinámico y ver el total actualizado en tiempo real.
  • Ventanas Modales: Se incluyen modales para el carrito y la confirmación de compra, lo que mejora la experiencia de usuario.
  • Productos Dinámicos: Los productos se generan dinámicamente, lo que permite una fácil actualización y gestión del catálogo.

Tecnologías Utilizadas:

  • HTML5 y CSS3: Para la estructura y el diseño del sitio, asegurando que sea responsive y accesible desde cualquier dispositivo.
  • JavaScript: Para manejar la lógica del carrito de compras, generación dinámica de productos y las ventanas modales.
